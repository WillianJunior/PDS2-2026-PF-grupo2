\doxysubsection{Detailed Description}
Representa um cômodo físico dentro de uma \doxylink{class_smarthome}{Smarthome}. 

Esta classe é responsável por gerenciar as condições do ambiente e agrupar logicamente objetos inteligentes, sensores, contas presentes e modos de automação.
